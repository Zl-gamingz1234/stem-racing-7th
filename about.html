About This Project

This website was created as a Class 7 STEM project to learn basic web design, HTML structure, and creative presentation. The goal was to combine science knowledge with technology skills.

Why We Chose This Logo

We chose this logo because it represents our team’s identity and legacy. We have been using this logo for the past two years, which makes it meaningful and familiar to our team. However, we also update and refine the logo each year to show the cycle of change and improvement. This balance between keeping the original idea and improving the design reflects how we learn, grow, and develop new skills over time. The logo is simple, recognizable, and represents both tradition and innovation.

What I Learned

  • Basic HTML and page structure
  • Design and color combinations
  • Uploading and hosting a website
  • Organizing files and folders

Future Improvements

In the future, I would like to add more interactive features, animations, and advanced styling.
